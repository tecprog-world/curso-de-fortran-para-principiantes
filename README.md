### CLONAR REPOSITORIO
Esto se puede ejecutar desde el Gitbash de Windows o desde la terminal de sistemas basados en UNIX.
```bash

```
### PARA ENVIAR LOS CAMBIOS AL REPOSITORIO
Abrir Gitbash o la Terminal y hacer.
```bash
$ git status	
$ git add --all
$ git commit -m "first commit"
$ git push
```
